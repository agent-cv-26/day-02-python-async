## 🛠 ملاحظات تقنية
- **Coroutines:**
- هي الدوال المعرفة بـ `async def` والتي تعيد "كائن كوروتين" بدلاً من القيمة مباشرة.
- **Await:** 
- هي نقطة التحرير؛ حيث تعيد التحكم إلى **حلقة الأحداث (Event Loop)** لخدمة مهام أخرى.
- **Gathering:**
- استخدام `asyncio.gather()` يسمح بتشغيل عدة مهام معاً بشكل متوازٍ منطقياً.
- **رؤية هندسية:** الـ Async ممتاز لعمليات الإدخال والإخراج (I/O) مثل طلبات الـ API وقواعد البيانات، لكنه يحتاج حذراً شديداً عند التعامل مع معالجة الصور (OpenCV) لأنها تستهلك المعالج (CPU-bound) وقد تعطل السيرفر إذا لم تُدار بشكل صحيح.
